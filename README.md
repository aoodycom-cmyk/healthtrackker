# Fat Loss Coach

Mobile-first fat loss tracker with:

- Calories and macros
- Cardio logging
- Weight and waist progress
- Body photo timeline
- USDA FoodData Central integration
- Barcode scanner
- AI Coach integration

## GitHub Pages

This folder is ready to publish as a static GitHub Pages site.

After pushing to GitHub:

1. Open repository Settings.
2. Go to Pages.
3. Choose Deploy from branch.
4. Select `main` and root `/`.
5. Open the generated HTTPS URL on mobile.

## API Keys

Do not commit real API keys.

The static app stores keys locally in the browser from:

`More > APIs`

For production, use a backend proxy for USDA and OpenAI keys.
